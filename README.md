# Meepo – Migrant Integration App

This was my final year university project.

The application helps migrants integrate into society by providing:
- job search
- language learning
- navigation
- weather information
- useful resources

Tech stack:
- React Native
- Node.js
- MongoDB
- OpenWeather API
- Google Maps API
